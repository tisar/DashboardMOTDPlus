DashboardMOTDPlus

by Daniel Obee @ 33achtzig

A simple module that allows to display the content of a standard response as Message of the Day in the OTRS dashboard.

How to install:
- install package via package manager
- create answer in admin interface as if used as email template
- got to Framework -> Frontend::Agent::Dashboard -> DashboardBackend###0090-MOTDPlus:
- set ResponseID to the id of the answer you want to display as MOTD
